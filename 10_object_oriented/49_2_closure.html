Understanding Closures in JavaScript

Example of Closure

      
        function createCounter() {
          let count = 0;
          return function() {
            count++;
            return count;
          };
        }

        const counter = createCounter();
        console.log(counter()); // 1
        console.log(counter()); // 2
        console.log(counter()); // 3

Explanation

A closure in JavaScript allows a function to retain access to its lexical scope, even when the function is executed outside its original scope.

In the example:

  • `createCounter` defines a `count` variable locally and returns an inner function.
  • The inner function (`counter`) is called multiple times, each time incrementing and returning `count`.
  • Despite `createCounter` finishing execution, `counter` retains access to `count` due to closure, allowing it to remember the state between calls.
