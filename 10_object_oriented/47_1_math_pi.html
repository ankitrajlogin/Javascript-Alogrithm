Properties of the Objects

Question : Can we Can the value of Pi in javascript

JavaScript's Math object is part of the standard built-in objects defined by ECMAScript. These properties are set up in a way that prevents accidental or intentional modification, which helps maintain consistency and avoids potential errors in calculations. Here's a brief explanation of property attributes in JavaScript:
  • Writable: If false, the property cannot be changed
  • Configurable: If false, the property cannot be deleted or changed (except for the writable attribute).
  • Enumerable: If true, the property will be included in enumerations (like in for...in loops).
Math.PI is defined with the following attributes:
  • Writable: false
  • Configurable: false
  • Enumerable: false 
console.log(Math.PI); // 3.141592653589793

Math.PI = 3.14;

console.log(Math.PI); // Still 3.141592653589793

The inability to overwrite Math.PI is due to its property attributes:

  • Writable: false
  • Configurable: false
  • Enumerable: false

These settings ensure the integrity and reliability of mathematical constants within the language. You can see these attributes using Object.getOwnPropertyDescriptor:

console.log(Object.getOwnPropertyDescriptor(Math, 'PI'));

This will output:

{
  value: 3.141592653589793,
  writable: false,
  enumerable: false,
  configurable: false
}
